# Technology Context

This project will use the following technologies to achieve its goals:

*   **Cloud Provider:** RunPod (for rapid, easy access to GPU compute).
*   **Infrastructure as Code:** Terraform
*   **Workload Orchestration (Compute):** Slurm
*   **Service Orchestration (Services):** Docker Compose
*   **Containerization:** Docker
*   **Backend:** Node.js with Express.js (TypeScript)
*   **Frontend:** React (TypeScript) with Vite and Ladle
*   **AI Agent:** LangChain.js
*   **Unit Testing:** Jest
*   **Observability:**
    *   NVIDIA DCGM Exporter (Metrics)
    *   Prometheus (Storage)
    *   Grafana (Dashboard)
*   **GPU Workload:** A simple Python script using PyTorch to stress the GPU.
